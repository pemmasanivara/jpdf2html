=== Build instructions ===

From the project root folder execute:

mvn clean package

=== Execution instructions ===

From command line: 
- navigate to <PROJECT HOME>\target\release
- make sure in the same folder there is a PDF named "test.pdf" (you can use the one provided in the root of the project)
- execute: jpdf2html.bat

The html file "generated.html" will be created. Open it with the browser of your choice.
